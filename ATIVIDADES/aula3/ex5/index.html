LINKS

Oque significa link em html ?

Link é um elemento que permite a ligação entre documentos.

  1. LINK INTERNO
      É um link para direcionamento para paginas internas
  2. LINK EXTERNO
      É um link para direcionamento para paginas externas
  3. LINK DE DOWNLOAD
      É um link para direcionamento para download de arquivos
  4. LINK LOCAL
      É um link para direcionamento para arquivos locais
